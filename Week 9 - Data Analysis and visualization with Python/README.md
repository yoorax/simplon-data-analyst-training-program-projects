# Week 9 – Netflix Data Analysis & Visualization

---

## Work Done

- **Exploratory Data Analysis (EDA)**
  - Executed a comprehensive Exploratory Data Analysis on a large, real-world dataset (`netflix_simplonline_dataset.csv`, 8,800+ records) using Python.

- **Data Cleaning & Profiling**
  - Cleaned and profiled the data using Pandas, identifying missing values across key columns (e.g., 30% missing in `director`, 9% in `cast` and `country`) and verifying data integrity (checking for duplicates and correct data types).

- **Key Metric Extraction**
  - Extracted key business metrics, computing the distribution of content types (Movies vs. TV Shows), the top 20 most popular genres, and the top 10 content-producing countries.

- **Data Visualization**
  - Designed and rendered professional data visualizations using Seaborn and Matplotlib, including countplots for content distribution and bar charts highlighting geographical production trends and release year concentrations.
