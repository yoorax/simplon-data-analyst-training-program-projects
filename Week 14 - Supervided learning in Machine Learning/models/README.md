# Machine Learning Models

The trained machine learning models for this project are too large to be hosted directly on GitHub.

You can download the model files from the following OneDrive link:

[Download Models from OneDrive](https://1drv.ms/f/c/6634fa2c11a9b1ac/IgB7lQvvo-2nTIuk9YZtDgSrAWULZUk6m30wwP48j36xISY?e=qKCcxr)

### Instructions
1. Download the `.joblib` files from the link above.
2. Place them in this `models/` directory.
3. Ensure your `.env` file points to this directory or the specific model files.

### Models Included:
- `random_forest_model.joblib`
- `rf_model_compressed.joblib`
- `xgb_model.joblib`
